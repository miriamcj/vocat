• Vocat API 1.0

Vocat is a teaching and learning platform that facilitates qualitative and quantitative assessment of a range of media. Built as a collaboration between the Center for Teaching and Learning at Baruch College, City University of New York and Cast Iron Coding of Portland, Oregon, the tool is in active development.

Resources

Annotations

Resource Description
GET /api/v1/annotations?asset=:asset returns all annotations for an asset
GET /api/v1/annotations/:id shows a single annotation
POST /api/v1/annotations creates a new annotation
PATCH /api/v1/annotations/:id updates an annotation
DELETE /api/v1/annotations/:id deletes a single annotation

Assets

Resource Description
GET /api/v1/assets/:id shows a single asset and its associated attachment, and annotations
POST /api/v1/assets creates a new asset
PATCH /api/v1/assets/:id updates an asset
DELETE /api/v1/assets/:id deletes an asset

Attachments

Resource Description
POST /api/v1/attachments creates a new attachment
GET /api/v1/attachments/:id shows a single attachment
DELETE /api/v1/attachments/:id deletes an attachment

Courses

Resource Description
GET /api/v1/courses returns all courses in which the current user is a member
GET /api/v1/courses/:id returns one course by ID
PATCH /api/v1/courses/:id updates a course

Discussion posts

Resource Description
GET /api/v1/discussion_posts?submission=:submission returns all discussion posts for a given submission
POST /api/v1/discussion_posts creates a discussion post for a submission
PATCH /api/v1/discussion_posts/:id creates a discussion post for a submission
DELETE /api/v1/discussion_posts/:id deletes a single discussion post

Enrollments

Resource Description
GET /api/v1/courses/:course_id/enrollments?role=:role returns users enrolled in a given course
GET /api/v1/users/:user_id/enrollments returns courses in which a given user is enrolled
DELETE /api/v1/enrollments/:enrollment_key removes a user from a course
POST /api/v1/enrollments enrolls a single user in a single course
POST /api/v1/courses/:course_id/enrollments/bulk bulk enrolls multiple users in a course

Evaluations

Resource Description
GET /api/v1/evaluations?submission=:submission returns all visible evaluations for a given submission
POST /api/v1/evaluations creates an evaluation
PATCH /api/v1/evaluations/:id updates an evaluation
DELETE /api/v1/evaluations/:id deletes an evaluation

Groups

Resource Description
GET /api/v1/groups?course=:course returns all groups for the given course
GET /api/v1/groups/:id shows a single group
POST /api/v1/groups creates a new group within a course
PATCH /api/v1/groups/:id updates a group
DELETE /api/v1/groups/:id deletes a group

Projects

Resource Description
GET /api/v1/projects?course=:course returns all projects for the given course
GET /api/v1/projects/:id shows a single project
POST /api/v1/projects creates a new project
PATCH /api/v1/projects/:id updates a project
DELETE /api/v1/projects/:id deletes a project
PATCH /api/v1/projects/:id/publish_evaluations publishes all of the current user's evaluations for the project
PATCH /api/v1/projects/:id/unpublish_evaluations unpublishes all of the current user's evaluations for the project

Rubrics

Resource Description
GET /api/v1/rubrics returns the current user's rubrics
GET /api/v1/rubrics/:id shows one rubric
POST /api/v1/rubric creates a new rubric
PATCH /api/v1/rubric/:id creates a new rubric
DELETE /api/v1/rubrics/:id deletes a rubric

Scores

Resource Description
GET /api/v1/scores/my_scores?project=:project returns the current user's scores for a given project
GET /api/v1/scores/all_scores?project=:project returns scores for a given project
GET /api/v1/scores/peer_scores?project=:project returns peer evaluation scores for a given project
GET /api/v1/scores/evaluator_scores?project=:project returns evaluator scores for a given project

Submissions

Resource Description
GET /api/v1/submission/for_course?course=:course returns summaries of all submissions for the given course
GET /api/v1/submission/for_creator_and_project?creator=:creator&project=:project&creator_type=:creator_type returns detailed submissions for the given creator and project
GET /api/v1/submission/for_course_and_user?course=:course&project=:project&user=:user returns detailed submissions for the given user, course, and project (optional)
GET /api/v1/submission/for_group?group=:group returns detailed submissions for the given group
GET /api/v1/submission/for_project?project=:project returns brief submissions the given project
GET /api/v1/submission/:id shows a single submission
POST /api/v1/course/:course_id/submissions DEPRECATED creates a submission
PATCH /api/v1/submissions/:id DEPRECATED updates a submission
DELETE /api/v1/submissions/:id deletes a submission and all attached media, annotations, discussion posts, and evaluations.

Users

Resource Description
GET /api/v1/users/search/?email=:email Finds users by email address. Under normal circumstances, should return an arry with 1 or 0 items.
GET /api/v1/users/:id shows a user
GET /api/v1/users/me shows the current authenticated user
