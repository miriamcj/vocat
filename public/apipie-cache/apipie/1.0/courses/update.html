PATCH /api/v1/courses/:id
updates a course

Examples

Sample Request:

{
  "id": 234,
  "department": "CIS",
  "description": "Updated description....",
  "name": "Updated Course Name",
  "number": "3810",
  "section": "X07SO",
  "organization_id": 1,
  "role": "evaluator",
  "semester_name": "Winter",
  "year": 2015
}

Errors

  • 403: The user is not authorized to update the course. Only admins and course evaluators/assistants may update courses.
