GET /api/v1/evaluations?submission=:submission
returns all visible evaluations for a given submission

This endpoint returns only those evaluations that are visible to the authenticated user. Furthermore, the authenticated user must have read access to the submission. Evaluators can see all evaluations for submissions within their courses. Creators can see all published evaluations on submissions they own. Peers can only see their own evaluations. Note that the exact fields returned in the response are dependent on what rubric is being used for the submission project.
