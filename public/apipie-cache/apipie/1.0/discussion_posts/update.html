PATCH /api/v1/discussion_posts/:id
creates a discussion post for a submission

Examples

Sample Request:

{
    "id": "1816",
    "body": "This is the UPDATED body of the post. _Markdown_ is *supported*"
}

Request Parameters

Param name Description
id
optional

The ID of the discussion post to be updated


Value:

Must be Fixnum
submission_id
optional

The ID of the submission being discussed


Value:

Must be Fixnum
parent_id
optional

If the post is a reply to another post, this is the ID of the parent post.


Value:

Must be Fixnum
body
optional

The body of the post, can be Markdown which will be converted to HTML when the post is rendered


Value:

Must be Fixnum
