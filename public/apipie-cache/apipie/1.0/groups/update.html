PATCH /api/v1/groups/:id
updates a group

Examples

Sample Request:
{
  "course_id": 234,
  "name": "Test Group 1",
  "creator_ids": [4499, 4506, 4503]
}

Errors

  • 403: The current user is not allowed to modify this group.
  • 404: A Group or User was not found.
  • 422: There was an error updating the group. Perhaps not all creators in the group are enrolled in the course?

Request Parameters

Param name Description
id
optional

The group's ID


Value:

Must be Fixnum
name
optional

The name of the group


Value:

Must be String
course_id
optional

The ID of the course to which the group belongs


Value:

Must be Fixnum
creator_ids
optional

An array containing the IDs of the users who belong to this group.


Value:

Must be an array of any type
