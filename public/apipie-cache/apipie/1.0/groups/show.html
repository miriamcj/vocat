GET /api/v1/groups/:id
shows a single group

Examples

Sample Response:

{
    "id": 810,
    "name": "Group #1",
    "creator_ids": [
        4483,
        4474,
        4520
    ],
    "course_id": 234,
    "first_name": "Group #1",
    "members": [
        "Betty Blick",
        "Shawn Jones",
        "Dewitt Ruecker"
    ]
}

Errors

  • 404: The group was not found.
  • 403: The current user does not have read access for the given course.

Request Parameters

Param name Description
id
required

The group ID


Value:

Must be Fixnum
