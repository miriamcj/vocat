Discussion posts

Discussion Posts appear on submission detail views. A post can be threaded, and can have another discussion posts as its parent. As with annotations, the author_id value of a discussion post is set to the user who creates the post. The API does not currently provide a mechanism for creating discussion posts that belong to other users.

>>>

GET /api/v1/discussion_posts?submission=:submission
returns all discussion posts for a given submission

Nota Bene: the posts are returned in a flat array. It is the responsibility of the client application to structure the posts into a parent/child hierarchy based on the parent_id.

POST /api/v1/discussion_posts
creates a discussion post for a submission

Examples

Sample Request:

{
    "submission_id": "7394",
    "body": "This is the body of the post. _Markdown_ is *supported*",
    "parent_id": null
}

Sample Response:

{
    "id": 1817,
    "author_id": 4470,
    "author_name": "Ressie Crona",
    "body": "<p>This is the body of the post. <em>Markdown</em> is <em>supported</em></p>",
    "body_raw": "This is the body of the post. _Markdown_ is *supported*",
    "published": null,
    "parent_id": null,
    "gravatar": "http://gravatar.com/avatar/4c3fe1bdae8ec9e5db97d02aa4cf1cda.png?d=mm&s=",
    "created_at": "2015-05-06T13:46:23.515-04:00",
    "month": "May",
    "day": "06",
    "year": "2015",
    "time": "01:46 PM",
    "current_user_can_reply": true,
    "current_user_can_destroy": true,
    "submission_id": 7394
}

Request Parameters

Param name Description
submission_id
required

The ID of the submission being discussed


Value:

Must be Fixnum
parent_id
optional

If the post is a reply to another post, this is the ID of the parent post.


Value:

Must be Fixnum
body
required

The body of the post, can be Markdown which will be converted to HTML when the post is rendered


Value:

Must be Fixnum
>>>

PATCH /api/v1/discussion_posts/:id
creates a discussion post for a submission

Examples

Sample Request:

{
    "id": "1816",
    "body": "This is the UPDATED body of the post. _Markdown_ is *supported*"
}

Request Parameters

Param name Description
id
optional

The ID of the discussion post to be updated


Value:

Must be Fixnum
submission_id
optional

The ID of the submission being discussed


Value:

Must be Fixnum
parent_id
optional

If the post is a reply to another post, this is the ID of the parent post.


Value:

Must be Fixnum
body
optional

The body of the post, can be Markdown which will be converted to HTML when the post is rendered


Value:

Must be Fixnum
>>>

deletes a single discussion post

Param name Description
