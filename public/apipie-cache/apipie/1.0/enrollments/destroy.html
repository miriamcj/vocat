DELETE /api/v1/enrollments/:enrollment_key
removes a user from a course

Only course evaluators/assistants and Vocat administrators may remove users from courses.

Errors

  • 422: Unable to remove the user from the course
  • 403: Authenticated user is not authorized to mange enrollment for this course.

Request Parameters

Param name Description
enrollment_key
required

The enrollment key is a string that takes the format of uXcY where X is the ID of the user to be de-enrolled, and Y is the ID of the course from which to de-enroll the user. If, for example, you wanted to remove User #10 from Course #234, the enrollment key would be u10c234


Value:

Must be String
