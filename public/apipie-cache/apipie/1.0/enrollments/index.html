GET /api/v1/courses/:course_id/enrollments?role=:role
returns users enrolled in a given course

GET /api/v1/users/:user_id/enrollments
returns courses in which a given user is enrolled

Only admins and evaluator users may see what courses a user is enrolled in. Any user who is enrolled in a course may view which other users are enrolled in that course.

Errors

  • 404: The requested course or user does not exist.
  • 403: The user is not authorized to view enrollments for this course or user.

Request Parameters

Param name Description
course_id
optional

The course's ID


Value:

Must be Fixnum
user_id
optional

The user's ID


Value:

Must be Fixnum
role
optional

Filter enrollments by course role; only available when getting enrollments by course. If this is unset, all enrollments for the course will be returned.


Value:

Must be one of: evaluator, creator, assistant.
