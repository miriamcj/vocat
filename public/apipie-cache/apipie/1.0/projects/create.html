POST /api/v1/projects
creates a new project

Examples

Sample Request:

{
  "project": {
     "name": "Test Project",
     "description": "Project description...",
     "course_id": 234,
     "type": "UserProject",
     "rubric_id": null,
     "allowed_attachment_families": ["image", "audio"],
     "due_date": "2015-04-23T18:25:43.511Z",
     "settings": {
        "enable_creator_attach": true,
        "enable_self_evaluation": false,
        "enable_peer_review": true,
        "enable_public_discussion": true,
        "reject_past_due_media": false,
        "anonymous_peer_review": true
     }
  }
}

Errors

  • 422: There was an error creating the project.

Request Parameters

Param name Description
project
required

The key here can be project, user_project, or group_project


Value:

Must be a Hash
project[type]
required

The type of Project


Value:

Must be one of: UserProject, GroupProject, OpenProject.
project[name]
required

The name of the project


Value:

Must be String
project[description]
required

Some information about the project


Value:

Must be String
project[course_id]
required

The ID of the course to which the project belongs


Value:

Must be Fixnum
project[rubric_id]
optional

If the project uses a scoring rubric, the ID of the rubric


Value:

Must be Fixnum
project[listing_order_position]
optional

The order of the project within the course


Value:

Must be Fixnum
project[allowed_attachment_families]
optional

The kinds of attachments that are accepted for this project; if an empty array, all types are accepted


Value:

Must be one of: audio, image, video.
project[due_date]
optional

The date after which no more assets may be attached to the project


Value:

Must be DateTime
project[settings]
required

Project settings


Value:

Must be a Hash
project[settings][enable_creator_attach]
optional , nil allowed

Whether or not the project allows creators to attach media


Value:

Must be one of: true, false.
project[settings][enable_self_evaluation]
optional , nil allowed

Whether or not the project allows self evaluation


Value:

Must be one of: true, false.
project[settings][enable_peer_review]
optional , nil allowed

Whether or not the project allows peer review


Value:

Must be one of: true, false.
project[settings][enable_public_discussion]
optional , nil allowed

Whether or not the project allows public discusion


Value:

Must be one of: true, false.
project[settings][reject_past_due_media]
optional , nil allowed

Whether or not the project should reject media after due date


Value:

Must be one of: true, false.
project[settings][anonymous_peer_review]
optional , nil allowed

Whetehr or not peer reviews are anonymous


Value:

Must be one of: true, false.
