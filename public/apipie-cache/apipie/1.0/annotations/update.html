PATCH /api/v1/annotations/:id
updates an annotation

Examples

Sample Request:

{
  "id" : 1,
  "body": "Another updated annotation...",
  "canvas": "{"json":null,"svg":null}",
  "published": true,
  "seconds_timecode": 11.143548,
  "smpte_timecode": "00:00:11",
  "asset_id": 4542
}

Errors

  • 404: The annotation does not exist.
  • 403: The user is not authorized to update the annotation.
  • 400: A required annotation parameter is missing.

Request Parameters

Param name Description
id
optional

The ID of the annotation to be updated


Value:

Must be a number.
body
optional

The body of the annotation; support markdown text


Value:

Must be String
canvas
optional

A JSON string that includes the json and svg representation of any drawings included in the annotation


Value:

Must be String
smpte_timecode
optional

A SMPTE formatted timecode (HH:MM:SS) pointing at the playback time that the annotation relates to


Value:

Must be String
published
optional

Whether or not the annotation is published [not currently used]


Value:

Must be one of: true, false.
seconds_timecode
optional

The value of this parameter should be the seconds at which the annotation occurs, to multiple decimal places. If the annotated asset does not have duration, this should be set to 0.0.


Value:

Must be Float
asset_id
optional

The ID of the asset that has been annotated


Value:

Must be Fixnum
