GET /api/v1/annotations/:id
shows a single annotation

Errors

  • 404: The annotation does not exist.
  • 403: The user is not authorized to view the annotation.

Request Parameters

Param name Description
id
optional

The ID of the annotation to be shown


Value:

Must be a number.
