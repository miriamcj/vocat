DELETE /api/v1/attachments/:id
deletes an attachment

Errors

  • 404: The attachment does not exist.
  • 403: The user is not authorized to delete the attachment.

Request Parameters

Param name Description
id
optional

The ID of the attachment to be deleted


Value:

Must be a number.
