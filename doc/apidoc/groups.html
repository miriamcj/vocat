Groups

Groups are collections of students that are tied to a course. Groups may create submissions for group projects and/or open projects.

POST /api/v1/groups
creates a new group within a course

Examples

Sample Request:
{
  "course_id": 234,
  "name": "Test Group 1",
  "creator_ids": [4499, 4506, 4503]
}

Sample Response:

{
    "id": 830,
    "name": "Test Group 1",
    "creator_ids": [
        4499,
        4506,
        4503
    ],
    "course_id": 234,
    "first_name": "Test Group 1",
    "members": [
        "Claude Erdman",
        "Nigel Gulgowski",
        "Wilhelmine Hackett"
    ]
}

Errors

  • 403: The current user is not allowed to create groups in this course.
  • 404: A Group or User was not found.
  • 422: There was an error creating the group. Perhaps not all creators in the group are enrolled in the course?

Request Parameters

Param name Description
id
optional

The group's ID


Value:

Must be Fixnum
name
required

The name of the group


Value:

Must be String
course_id
required

The ID of the course to which the group belongs


Value:

Must be Fixnum
creator_ids
required

An array containing the IDs of the users who belong to this group.


Value:

Must be an array of any type

Examples
