POST /api/v1/rubric
creates a new rubric

Examples

Sample Request:
  {
      "name": "My Rubric",
      "description": "Desc...",
      "low": 0,
      "high": 10,
      "ranges": [
          {
              "id": "low",
              "name": "Low",
              "low": 0,
              "high": 5
          },
          {
              "id": "high",
              "name": "High",
              "low": 6,
              "high": 10
          }
      ],
      "fields": [
          {
              "name": "Voice",
              "id": "voice",
              "description": "Voice Description"
          },
          {
              "name": "Body",
              "id": "body",
              "description": "Body Description"
          }
      ],
      "cells": [
          {
              "range": "low",
              "field": "voice",
              "description": "low voice"
          },
          {
              "range": "low",
              "field": "body",
              "description": "low body"
          },
          {
              "range": "high",
              "field": "voice",
              "description": "high voice"
          },
          {
              "range": "high",
              "field": "body",
              "description": "high body"
          }
      ]
  }

Errors

  • 422: There was an error creating the rubric.
