POST /api/v1/annotations
creates a new annotation

Examples

Sample Request:

{
  "body": "Another annotation...",
  "canvas": "{"json":null,"svg":null}",
  "published": true,
  "seconds_timecode": 11.143548,
  "smpte_timecode": "00:00:11",
  "asset_id": 4542
}

Sample Response:

{
    "id": 11680,
    "asset_id": 4542,
    "author_id": 4470,
    "body": "<p>Another annotation...</p>",
    "published": true,
    "seconds_timecode": 11.143548,
    "smpte_timecode": "00:00:11",
    "author_name": "Ressie Crona",
    "canvas": "{"json":null,"svg":null}",
    "current_user_can_destroy": true,
    "current_user_can_edit": true,
    "gravatar": "http://gravatar.com/avatar/4c3fe1bdae8ec9e5db97d02aa4cf1cda.png?d=mm&s=",
    "author_role": "evaluator",
    "created_at": "5/05/15 07:49PM",
    "created_at_timestamp": 1430869781,
    "body_raw": "Another annotation..."
}

Errors

  • 403: The user is not authorized to annotate the asset to which the annotation belongs.
  • 400: A required annotation parameter is missing.

Request Parameters

Param name Description
body
required

The body of the annotation; support markdown text


Value:

Must be String
canvas
optional

A JSON string that includes the json and svg representation of any drawings included in the annotation


Value:

Must be String
smpte_timecode
optional

A SMPTE formatted timecode (HH:MM:SS) pointing at the playback time that the annotation relates to


Value:

Must be String
published
optional

Whether or not the annotation is published [not currently used]


Value:

Must be one of: true, false.
seconds_timecode
required

The value of this parameter should be the seconds at which the annotation occurs, to multiple decimal places. If the annotated asset does not have duration, this should be set to 0.0.


Value:

Must be Float
asset_id
required

The ID of the asset that has been annotated


Value:

Must be Fixnum
