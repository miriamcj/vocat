GET /api/v1/scores/my_scores?project=:project
returns the current user's scores for a given project

Examples

Sample Response:

{
    "summary": {
        "project": {
            "asset_count": 0,
            "possible_submission_count": 13,
            "rubric_avg_score": "62.0",
            "rubric_avg_percentage": "68.8888888888889"
        },
        "score": {
            "average_score": 0,
            "average_percentage": 0,
            "count": 0
        }
    },
    "scores": []
}

Request Parameters

Param name Description
project
required

The project ID


Value:

Must be Fixnum
