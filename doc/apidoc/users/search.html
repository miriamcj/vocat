GET /api/v1/users/search/?email=:email
Finds users by email address. Under normal circumstances, should return an arry with 1 or 0 items.

Examples

Sample Response:
[
  {
    "id": 4470,
    "email": "evaluator1@test.com",
    "name": "Ressie Crona",
    "gravatar": "http://gravatar.com/avatar/4c3fe1bdae8ec9e5db97d02aa4cf1cda.png?d=mm&s=",
    "first_name": "Ressie",
    "last_sign_in_at": "2015-05-06T17:51:54.327-04:00",
    "org_identity": "30683835",
    "list_name": "Crona, Ressie"
  }
]

Request Parameters

Param name Description
email
optional

The user's email address


Value:

Must be String
